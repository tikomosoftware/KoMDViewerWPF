# KoMDViewer WPF

Windows向けの軽量Markdownビューアです。

## 必要なランタイム

- .NET 9 Desktop Runtime
- Microsoft Edge WebView2 Runtime

ZIPに同梱されている `install-runtime.bat` を実行すると、必要なランタイムの確認と案内を行います。

## 使い方

1. ZIPを任意のフォルダに展開します。
2. 初回のみ `install-runtime.bat` を実行します。
3. `KoMDViewerWPF.exe` を起動します。

Markdownファイルは、メニューの `ファイル > 開く...`、`Ctrl + O`、またはドラッグ&ドロップで開けます。

## 主な機能

- Markdown / text file viewer
- 左側の履歴ペイン
- 履歴からの削除
- 履歴のファイル名順ソート
- ドラッグ&ドロップ
- ライト / ダークテーマ

## 対応ファイル

- `.md`
- `.markdown`
- `.mdown`
- `.mkd`
- `.mkdn`
- `.txt`
